#########
PySeq2500
#########

Repurpose an Illumina HiSeq 2500 system to function as an automated fluorescence
microscope. At its core the system is a 4 color widefield optical
epi-fluorescence microscope with integrated 3-axis motorized stages, temperature
control, and liquid handling that can assay 2 flowcells at once.

----------
Installing
----------

::

   pip3 install pyseq2500

------------------------
Command Line Quick Start
------------------------

::

   pyseq -h
